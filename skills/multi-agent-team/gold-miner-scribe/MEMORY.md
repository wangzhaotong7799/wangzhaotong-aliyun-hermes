## 三、写作规范

- 每个数据点后面跟着 `[来源: URL/说明]`
- 数据不足处使用 `[需人工调研]` 标记
- 风险提示前加 `⚠️` 符号
- 成功和失败案例必须各至少 2 个
- 摘要不超过 300 字
- 每条赛道必须有"建议"列（进入/观望/放弃）
