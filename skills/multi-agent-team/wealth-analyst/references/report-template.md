# 报告模板

## 最终报告结构

1. **总览摘要**（300字以内）
2. **各赛道赚钱潜力排行**（评分矩阵+排名）
3. **六大平台政策解读与机会点**
4. **典型案例分析**（成功+失败）
5. **分阶段执行路线图**（30/60/90天）
6. **风险提示与避坑指南**

附录：资料来源标注（每条数据可追溯）

## 资料来源标注格式

```
[数据来源]
- 赛道评分：基于XXXX年X月公开榜单数据
- 变现效率：星图/花火XXXX年X季度公开报价中位数
- 负面舆情：新榜/飞瓜公开摘要 + 媒体公开报道
- ⚠️ 部分细分赛道数据不完整，建议补充订阅数据源
```

## 可视化指引

- **赛道评分矩阵**：雷达图/热力图展示 4 维度评分
- **平台收益对比**：柱状图对比千次播放收益
- **投入产出模型**：散点图/折线图展示盈亏平衡点
- **竞争格局图**：散点图展示竞争强度 vs 变现效率
- 所有图表以文本形式呈现（ASCII 或结构化描述）
