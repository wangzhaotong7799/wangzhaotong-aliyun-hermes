# 金脉小队 — 域适配指南

将金脉小队从自媒体域迁移到其他调查域的配置手册。

---

## 域参数表

| 参数 | 说明 | 示例 |
|------|------|------|
| 域（Domain） | 调查的大类名称 | 自媒体、电商、本地生活、AI应用 |
| 赛道（Tracks） | 域下的细分方向列表 | [直播电商, 跨境电商, 拼多多白牌] |
| 数据平台 | 数据来源平台/渠道 | [抖音, 小红书, B站] vs [淘宝, 京东, 拼多多] |
| 评分维度 | 算盘使用的评分框架 | 见各域的「评分维度」节 |
| 章节结构 | 报告模板的章节 | 见各域的「报告章节」节 |
| 典型案例池 | 成功/失败案例的关键词 | [电商倒闭, 直播翻车, 品牌崩盘] |

## 域案例一：自媒体（已验证 ✅）

**首次运行验证通过**（2025年5月2日），产出 180 行/17.5KB 专业报告。

```
域：自媒体
赛道：美妆、数码、教育、母婴、短剧、本地生活、健身、美食
数据平台：抖音、小红书、B站、视频号、快手、TikTok
评分维度：变现效率(0.35) + 竞争强度(0.25) + 供给饱和度(0.25) + 生命周期(0.15)
报告章节：6章（摘要→排行→平台→案例→路线图→风险）
失败案例关键词："自媒体 亏了"、"MCN 倒闭"、"达人 翻车"、"平台 纠纷"
数据引擎：Firecrawl(深爬) + Exa(语义) + Tavily(实时)
```

## 域案例二：电商 （已验证 ✅ — 2025年5月3日）

**10 个赛道配置：**

```
域：电商
赛道（10个，按生态分组）：
  国内平台：
    1. 国内电商平台（淘宝/天猫/京东）— 全品类大盘
    2. 拼多多白牌 — 产业带直供/性价比战场
    3. 快手电商 — 新线城市白牌主阵地，GMV超万亿
    4. 视频号电商 — 银发经济红利，DAU 6.5亿
  内容电商：
    5. 直播电商（抖音电商）— 内容驱动，GMV增速最快
    6. 社交电商（小红书/微信私域）— 种草+信任成交
  跨境电商：
    7. TikTok Shop — 内容电商出海，美区+东南亚
    8. TEMU 全托管 — 极致性价比，90+国家
    9. 东南亚电商（Shopee/Lazada）— 海外增量市场
  新形态：
    10. 即时零售（美团闪购/京东到家）— 近场电商吃掉传统份额

数据平台（按赛道分组）：
  - 国内平台：淘宝/天猫/京东、拼多多 → web_search 搜行业报告+榜单
  - 内容电商：抖音电商、快手电商、小红书 → web_search + Exa 语义搜索
  - 跨境电商：TikTok Shop、TEMU、Shopee → Exa 搜行业报告，Tavily 搜实时政策
  - 新形态：美团闪购、京东到家 → web_search 第三方分析

评分维度（电商域专用权重）：
  - 市场规模增速（0.30）— 赛道年增长率
  - 竞争强度（0.25）— 头部玩家集中度
  - 供应链复杂度（0.20）— 入驻门槛/运营难度
  - 生命周期阶段（0.15）— 爆发期/成长期/成熟期
  - 政策风险（0.10）— 合规/关税/反垄断

报告章节（电商域）：
  1. 总览摘要 + 核心结论
  2. 电商赛道赚钱潜力排行（评分矩阵表）
  3. 各平台政策与机会点（淘宝/京东/拼多多/抖音/快手/TikTok Shop/TEMU）
  4. 典型案例（成功+失败：东方甄选、交个朋友、拼多多白牌起量、TikTok Shop卖家故事）
  5. 分阶段执行路线图（30/60/90天）
  6. 风险提示（流量成本、平台抽成、关税政策、合规红线）

失败案例搜索关键词：
  - 「电商 倒闭」「直播 翻车」「拼多多 崩盘」
  - 「TikTok Shop 亏了」「TEMU 压榨商家」「跨境 库存积压」

LTV 预估方式：
  - 客单价 × 复购率 × 留存月数（区别于自媒体的播放量×RPM）
  - 电商域典型 LTV 区间参考：白牌 50-300元 / 直播 100-500元 / 跨境 200-1000元

## 域案例五：CPS联盟营销（新域 ✅ — 2026年5月2日）

```
域：CPS联盟营销
赛道（6个）：
  1. 外卖CPS（美团/饿了么返利）— 刚需高频，佣金薄但量大
  2. 电商CPS（淘宝客/京东联盟）— 最成熟赛道，但红利消退中
  3. 本地生活CPS（电影票/打车/到店）— 高客单低频，利润空间大
  4. 社交裂变CPS（微信群/公众号/私域）— 核心壁垒在于"人"，非标准化
  5. 短视频带货CPS（抖音/快手分佣）— 内容+佣金双驱动，增速最快
  6. 跨境CPS（TikTok Shop affiliate）— 增长最快，政策风险最高

数据平台：
  - 外卖CPS → 美团联盟、饿了么联盟
  - 电商CPS → 淘宝联盟、京东联盟、多多进宝
  - 本地生活CPS → 美团联盟、猫眼/淘票票、滴滴
  - 社交裂变CPS → 微信群/公众号、返利网、什么值得买
  - 短视频带货CPS → 抖音精选联盟、快手快分销
  - 跨境CPS → TikTok Shop Affiliate、Amazon Associates
  引擎策略：佣金数据需"行业报告+实测交叉验证"，Exa用于政策搜索，Tavily用于实时动态

评分维度（CPS专属权重）：
  - CPS佣金率/利润率（0.25）— 各赛道每单实际到手佣金比
  - 用户生命周期LTV（0.25）— 复购频次 × 人均留存周期
  - 平台政策稳定性（0.20）— 佣金下调/封禁/准入门槛
  - 市场容量（0.20）— 可触达用户规模天花板
  - 竞争强度（0.10）— 头部玩家集中度+新玩家入场门槛

报告章节（CPS域）：
  1. 摘要 + 核心结论 — 6赛道横向对比总览
  2. 赛道赚钱潜力排行（评分矩阵表，含佣金率实测估算）
  3. 各平台CPS政策对比（佣金比例/规则/准入条件）
  4. 案例池 — 成功：省钱快报、值得买、微信群裂变矩阵
               失败：政策封禁、佣金断崖式下调、微信生态收紧
  5. 入局路线图（按资源禀赋分三档：个人→小团队→机构）
  6. 风险清单（平台抽成下调风险、封号、反垄断、微信生态收紧）

失败案例搜索关键词：
  - 「返利 崩了」「CPS 被封」「淘宝客 佣金下调」
  - 「美团联盟 门槛」「微信群 裂变 封禁」「带货 翻车 佣金」

LTV 预估方式：
  - CPS域特殊：单用户ARPULTV = 平均月返利额 × 留存月数
  - CPS的LTV特点：用户粘性来自"省钱刚需"，比纯电商更高但受平台政策冲击大
  - 个人CPS典型LTV区间：外卖用户10-50元/月 / 电商返利用户20-100元/月 / 本地生活50-200元/单

数据采集注意：
  佣金比例数据难以从公开搜索完整获取，优先依赖：
  (1) 行业报告中的佣金范围参考
  (2) 各平台联盟官网公开规则
  (3) 实测/社群调研（标注[需实测]）
  禁止编造佣金数据，无公开数据处标注"公开数据不足"
```

## 域适配决策树

```
用户说"调查YY" → 判断域属于哪类
  ├─ 内容/流量导向 → 自媒体域（已验证）
  ├─ 商品/交易导向 → 电商域
  ├─ 线下/服务导向 → 本地生活域
  ├─ 工具/软件导向 → AI应用域
  └─ 佣金/返利/分佣导向 → CPS联盟营销域（新）

确认域后 → 查上表确定：
  1. 赛道列表
  2. 数据平台组合（写进天网指令）
  3. 评分维度权重（写进算盘指令）
  4. 章节结构（写进执笔指令）
  5. 失败案例搜索词（写进军师指令）
```

## 如何新增一个域

1. 在本文件底部添加新的 `## 域案例X：<域名称>` 节
2. 必须包含：域参数表（赛道、数据平台、评分维度权重、报告章节、失败案例关键词、LTV预估方式）
3. 同时确认该域已有足够的公开数据可用（如果主要数据来源涉及付费 API，备注说明）
4. 更新后无需修改 SKILL.md，猎财自动读取本文件
5. 创建对应 cron job 测试是否跑通

---

## 重要注意事项

1. **无数据时不要编造** — 某个域没有足够公开数据时，标注"数据不足"并建议付费数据源
2. **按域调整预期** — 自媒体数据最丰富（6大平台×8赛道），电商次之，本地生活线下部分最难
3. **三引擎切换** — Firecrawl 适合深爬已知页面，Exa 适合语义搜索行业报告，Tavily 适合实时热点
4. **多域联动** — 一个域内的赛道可能跨域（如"直播带货"横跨电商+自媒体），按主要域归类即可
