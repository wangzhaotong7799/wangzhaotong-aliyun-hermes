## 六、评分模型参数

### 赛道评分维度（0-100）

```python
SCORE_MODEL = {
    "变现效率": {
        "权重": 0.35,
        "数据源": "星图/花火公开报价、报道GMV区间",
        "评分逻辑": "报价中位数越高、GMV区间上沿越高 → 分数越高"
    },
    "竞争强度": {
        "权重": 0.25,
        "数据源": "头部账号粉丝集中度、新入场破10万账号数",
        "评分逻辑": "集中度越低、新入场者越多 → 分数越低（竞争激烈）→ 倒序计分"
    },
    "内容供给饱和度": {
        "权重": 0.25,
        "数据源": "每日新增作品量 / 日均互动总量",
        "评分逻辑": "比值越低 → 供给缺口大 → 分数越高"
    },
    "生命周期阶段": {
        "权重": 0.15,
        "数据源": "定性判断（萌芽/爆发/成熟/衰退）",
        "评分逻辑": "萌芽=100, 爆发=80, 成熟=50, 衰退=10"
    }
}
```

### 盈亏平衡点计算公式
```
千次播放收益（RPM）× 有效播放量 = 总制作成本
→ 有效播放量下限 = 总制作成本 / RPM
```
