## 四、失败案例与风险信号

1. **2024电商"死亡"名单暴增近400%** — 同比暴增，行业洗牌加剧
2. **TEMU关税冲击** — 取消800美元免税致短期销售额降32%
3. **跨境电商大洗牌** — 迅达电器、永生电器等老牌跨境公司相继解散
4. **跨境"四小龙"分流** — Temu从激进转向治理，美国广告投入-54%
5. **老牌零售关店潮** — 中百集团关30店，预计损失1.8亿
6. **盒马战略教训** — 关停部分门店，战略懒惰代价
7. **直播电商中小主播困境** — 虽然占比89%，但头部效应明显，新入局者需精细运营
