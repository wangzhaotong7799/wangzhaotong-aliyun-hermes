## 二、评分模型参数

```python
SCORE_WEIGHTS = {
    "变现效率": 0.35,
    "竞争强度": 0.25,
    "内容供给饱和度": 0.25,
    "生命周期阶段": 0.15,
}

LIFECYCLE_MAP = {
    "萌芽期": 100,
    "爆发期": 80,
    "成熟期": 50,
    "衰退期": 10,
}
```

### 评分逻辑

- **变现效率**: 官方报价中位数 + 场均GMV区间上限，归一化到 0-100
- **竞争强度**: 倒序计分。头部集中度越低、新入场者越少 → 分越高
- **供给饱和度**: 每日新增作品/日均互动，比值越低（供给缺口大）→ 分越高
- **生命周期**: 定性判断，萌芽最高、衰退最低

### 盈亏平衡公式

```
保底播放量 = 总制作成本 / 千次播放收益(RPM)
```
